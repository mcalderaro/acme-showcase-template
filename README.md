# ACME ClawWorks Showcase

This repository powers the public documentary site for ACME ClawWorks.

ACME ClawWorks is a radically transparent AI-assisted engineering studio built with OpenClaw.

The goal of this project is to demonstrate how AI agents can collaborate to run a small software company, including:

- product planning
- architecture
- engineering
- QA
- milestone tracking
- public storytelling

Visitors can watch the company operate in real time through milestone articles, dev logs, and links to the actual work tracked in Plane.
